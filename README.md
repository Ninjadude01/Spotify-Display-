# Spotify-Display-

My project is a spotify display, where you can skip songs, pause/play, and go to the previous song! You do this by pressing they keyboard switches at the top, and when you do that, a piezo buzzer inside will buzz and produce different sounds depending on which keyboard switch you pressed. Inside, there is an esp32 that controls everything, and is connected to a small tft display, a piezo buzzer, and the three switches. There is also a breadboard inside. I decided to make this project as I love listening to music, and having a small device to control what is being played sounded interesting to me :)

No BOM is included as I have all the materials I need
